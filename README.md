# Vim

A minimal Vim key-binding emulation WebExtension that just works.

## Key bindings

|Key strokes| Function |
|---|---|
|`gg`| Go Top |
|`G`| Go Bottom |
|`j` | Scroll down |
|`k` | Scroll up |
|`f` | Enter Fullscreen |
|`r` | Refresh page |
|`b`| Go to site root |
